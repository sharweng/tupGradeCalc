# TUP Grade Calculator

This is a simple GPA calculator for TUP students. It allows you to:

- Add subjects with units and grades
- Lock grades for subjects you already know
- Calculate your current GPA
- Calculate the minimum grades needed to reach a target (President's Lister, Dean's Lister, Passed)
- Save and load your data automatically

## Usage

1. Add subjects and enter their units and grades.
2. Lock grades for subjects you already know.
3. Click "Calculate GPA" to see your current GPA.
4. Select a goal and click "Calculate for Goal" to see the minimum grades needed for unlocked subjects.
5. Use "Clear All" to reset the calculator.

## Requirements

- Modern web browser
- No installation required

## Notes

- All data is saved locally in your browser.
- Make sure to fill in all required fields before calculating.
